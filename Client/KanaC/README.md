## アルゴリズム

1. 石nに`((N - n) / N) * (ブロック数)`というスコアを振る
2. スコアが一番高い石を最初に置く
  1. その石より後の石で、スコアが一番高い石を次に置く
  2. その石より後の石で...
  3. もう置ける石が無い場合終わり
3. 「残り領域」とか「石の数」を覚えておく
4. 2. を「2番目にスコアが高い石」でやってみる
5. 「残さないで」、「石の数が少ない」のが良いとする

恐らく、以下が問題となる

* 石の配置について何も考えていない
* 最初の石以外の選び方が不十分
